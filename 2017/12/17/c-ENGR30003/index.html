6 Differentiation, differential equations

Write a C program that solves the heat equation
\begin{equation}
\dfrac{\partial{f}}{\partial{t}}=\mu \dfrac{\partial^2{f}}{\partial{x^2}}
\label{eq:10}
\end{equation}
on the interval (x=[0,1]), and for times (0 \le t \le 2.0), using the diffusion coefficient (\mu=0.005).

Most transport equations can also be written as
\begin{equation}
\dfrac{\partial{f}}{\partial{t}}=RHS(f)
\label{eq:11}
\end{equation}

where RHS denotes the ‘right-hand-side’ of the equation to be solved, containing all spatial derivatives, in this case (\mu \dfrac{\partial^2{f}}{\partial{x^2}}).

The heat equation is to be solved on the spatial interval (0 \le x \le 1), that is discretized using (N_x) points (x_i), with (i=0,1,2,\dots,N_x). The equidistant grid spacing therefore is (\Delta{x}=1/N_x).

In order to discretize the time derivative (\frac{\partial{f}}{\partial{t}}), two different time integration schemes are to be
used, an explicit Euler and an implicit Euler scheme, as follows

\begin{equation}
f_i^{n+1}=f_i^n+\Delta{t} \cdot RHS(f_i^n)\text{ (explicit)}\\
f_i^{n+1}=f_i^n+\Delta{t} \cdot RHS(f_i^{n+1})\text{ (implicit)}
\label{eq:12}
\end{equation}

where (n) is the time index so that (f_i^n) is the function value at time level (n) at point (x_i) and (f_i^{n+1}) is the function value at time level (n+1) at point (x_i), and (\Delta{t}) is the numerical timestep for the time integration.
